Barbaric Freefall



Barbaric Freefall is a platform avalanche game developed as a final project by a colleague. The game was created using Pixi.js, utilizing JavaScript, HTML, and CSS.



Role:

As the Technical Artist, I was responsible for the art assets of the game as well programming them to be functionable and smooth. I integrated...

  • Animation Code for Sprite Sheets
  • Randomly Generated Threats (Picking which sprite to use for the threat that will be spawned)
  • Threat Art Assets and Animations/Sprite Sheets (Shield, Axe and Arrow)
  • Powerup Art Assets (Speed Potion and Revive Potion)
  • Background Art Asset
  • Buttons Art Assets
  • Player Art Assets and Animations/Sprite Sheet
  • Player Movement Code (Keyboard Controls)
  • Scene Management

ColoRun



ColoRun is an endless color matching runner. Match colors while sliding in the correct direction to keep from getting caught by the disappearing tiles underneath you.



Highlighted Features

  • Quasi Random Generated Tiled Field: Player always solves a different puzzle
  • Color Impaired Option: Developed to be accessible to as many individuals as possible.
  • PowerUps: 3 game changing powerups adding new mechanics

Play Here / Read More

How to Play:

Quickly and precisely swap between four distinct colors (Red = “A”, Blue = “W”, Yellow = “D”, and Green = “F”) to slide between randomly generated tiles (Using Arrow Keys) and away from the ones disappearing underneath you.

  • Software: Unity Engine | Photoshop | Pyxel | Visual Studio
  • Language: C#
